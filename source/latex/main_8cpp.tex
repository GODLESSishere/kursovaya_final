Основной цикл для обработки сетевых операций. Эта функция запускает сетевой сервис и входит в бесконечный цикл, в котором она ожидает входящих соединений, аутентифицирует пользователей и выполняет операции. Любые сетевые исключения обрабатываются и регистрируются. Если возникает неожиданное исключение, сетевой сервис останавливается.

Главная точка входа в приложение. 

Эта функция инициализирует объект \mbox{\hyperlink{classDecoder}{Decoder}} для парсинга аргументов командной строки, получает путь к журналу и базу данных из указанной конфигурации, инициализирует объект \mbox{\hyperlink{classNetwork}{Network}} с этими настройками и входит в цикл для обработки сетевых операций. Также включает обработку ошибок для сетевых исключений.
